# Toadz Shop
 
  Apparel Store
 

 

# 


### Pre-requisite
```
install Nodejs 14

install MongoDB Community Edition

```


### Development commands
```
npm install
npm run server-dev  (in terminal 1 for api service)
npm run dev  (in terminal 2 for live-watched web server)
```


### Run Tests
```
npm install
npm run test  
```



### Packaging commands
```
npm run build
npm run server
```


## using pm2

 pm2 start pm2.config.json --env production 
pm2 monit
